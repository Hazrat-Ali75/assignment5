Question-2: What is the difference between innerHTML, innerText, and textContent?

1.innerHTML: HTML markup (tags and text). Use for HTML manipulation. Potential security risk.
2.innerText: Visible text, respects CSS styling. Use for user-displayed text. Slower.
3.textContent: Raw text content, ignores CSS. Use for getting all text. Faster.

Question-3: What is event delegation in the DOM?

Imagine you have a bunch of kids (child elements) and you want them to raise their hands (trigger an event). Instead of asking each kid individually to raise their hand, you tell them all: "When anyone raises their hand, I'll check who did it."
That's event delegation:
One listener: You put one "listener" (like your instruction) on the parent (the group of kids).
Check who: When a child triggers an event (raises their hand), you find out which child did it.
Easy for many: It's easier than telling each child separately, especially if you get more kids later.

Question-4: What is event bubbling in the DOM?

Imagine a bunch of nested boxes. If you tap the smallest box inside, the "tap" travels outwards, like ripples in water, through each bigger box surrounding it. That's event bubbling: an event on one element "bubbles up" to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Create:
document.createElement('div') - Creates a new div element.
Add:
parentElement.appendChild(newElement) - Adds newElement as a child to parentElement.
parentElement.insertBefore(newElement, referenceElement) - Inserts newElement before referenceElement.
Remove:
elementToRemove.remove() - Removes the element.
parentElement.removeChild(elementToRemove) - Removes elementToRemove from parentElement.
